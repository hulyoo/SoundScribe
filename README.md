# SoundScribe - 听歌扒谱工具

> AI-powered music transcription app that converts audio to sheet music

## 项目概述

SoundScribe 是一款跨平台的音乐扒谱应用，用户可以通过麦克风录音，AI 自动识别歌曲并生成吉他谱，最终导出 PDF 格式。

## 核心功能

- 🎤 麦克风实时录音
- 🎵 AI 智能扒谱（Melody.ml）
- 📄 PDF 吉他谱导出
- 🌐 跨平台支持（Windows/macOS/Linux/iOS/Android）

## 技术栈

- **框架**: Flutter
- **AI 服务**: Melody.ml
- **PDF 生成**: pdf package
